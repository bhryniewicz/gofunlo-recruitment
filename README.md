# Project documentation

Aby wlaczyć projekt musisz:

Jezeli masz wersje node > v18

```bash

nvm install v18.18.0

nvm use v18.18.0

```

Następnie zainstalować wszystkie paczki:

```bash
npm install
```

Aby uruchomić projekt:

```bash
npm run dev
```

Aby uruchomic testy:

```bash
npm run test
```
